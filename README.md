# dermo_project

Projemizin ilk aşamasında elimizdeki verilere görüntü iyileştirme tekniklerini uygulamak olacaktır.



Totalde 5 farklı tekniğimiz var.



Veriler içinde ayırt edilmesi güç olanlara bir teknik seçilerek uygulanacak.



Örneğin, egzama hastalığından bir foto seçildi buna histogram uyguladık, başka seçildi ona eşikleme uyguladık gibi..








Tüm işlemler bitince etiketleme adımına geçilecektir.
